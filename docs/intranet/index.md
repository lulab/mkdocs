## Network

### 实验室内上网

目前实验室分校园网（外网：166.156.。。。）及内部网（172.22.220….）。

每个座位都有标签标明外网、内网。 

> 每间屋子都有无线信号覆盖（接内网），密码联系计算机管理员获得。


### 校内上网

推荐使用 Tsinghua-Secure SSID 访问校园网，访问该无线网络采用的密码需与校园网账号密码不同,请提前在 <https://usereg.tsinghua.edu.cn> 中进行密码修改（从页面中“用户”下的“修改Tsinghua-secure密码”中进行修改，修改后的密码仅用于 Tsinghua-Secure SSID）。对于大部分终端，连接该 SSID 后，直接输入用户名和密码即可。对于 Windows7 等旧系统，需通过客户端配置上网，配置说明详见无线校园网802.1x（Tsinghua-Secure）认证登录配置说明。

连接Tsinghua、Tsinghua-5G、 Tsinghua-IPv4、Tsinghua-IPv6等无线网SSID 后，采用与有线网相同的认证机制。打开浏览器时，系统会自动弹出校园网认证系统登录页面，或在浏览器中输入认证系统网址 <https://login.tsinghua.edu.cn> 进行认证操作；不再使用网络时，需再次访问认证系统“断开连接”。

> 更多帮助：[https://its.tsinghua.edu.cn](https://its.tsinghua.edu.cn) -- **校园网络**
